For Matlab:
Change directory to ‘MATLAB_code’ folder and run the problem’s corresponding code.




For Python
Change directory to ‘Python_code’ folder and run corresponding code. 
This script runs on Python 2.7. 

Please make sure you have the following:



Needed Packages:
-matplotlib
-numpy
-scipy



Needed Files:
BrakingData.txt



Make sure all files are in the SAME directory or python will not be able to find them.


LaTex must also be installed on the machine and accessible to Python for the rendering of the plots.
